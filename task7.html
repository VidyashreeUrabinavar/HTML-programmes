Performed Poetry Using <pre>

Code & Dreams

In the silence of the midnight screen,
Lines of logic glow serene.
Ideas rise and gently flow,
Through loops that learn and functions grow.

Errors fade, the bugs retreat,
As patience makes the code complete.
Whispers of syntax, soft yet clear,
Turn simple thoughts to software here.

The <pre> tag preserves spaces and line breaks exactly as written, while formatting tags like bold, italic, highlight, emphasis, and small text still work inside it.
